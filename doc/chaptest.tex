\minitoc

\section{Introduction}

Dans bin/tests est disposé un fichier options.db3. Si on le copie un
directory en dessous ``cp options.db3 ../.'', quand on lance le code
après un load il apparaît quatre configurations test qui permettent de
voir toutes les options en action.

\section{Test1}

Le but du test1 est de tester un cas simple mais avec une précision
importante pour l'inversion du système d'équations linéaires 
(tolérance=$10^{-9}$) et de nombreuses options du code afin de les valider.
La Fig.~\ref{test1conf} montre les options de la configuration choisie.
